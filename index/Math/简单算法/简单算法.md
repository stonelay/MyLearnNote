
## 二分查找

### 算法要求
1. 必须采用顺序存储结构。
2. 必须按关键字大小有序排列。

### 算法复杂度
log2n

### code
```C
int bsearchWithoutRecursion(int array[],int low,int high,int target) {
    while(low<=high) {
        int mid=(low+high)/2;
        if(array[mid]>target)
            high=mid-1;
        else if(array[mid]<target)
            low=mid+1;
        else
            return mid;
    }
    return -1;
}
```
